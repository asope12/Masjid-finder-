## 📱 Overview

Masjid Finder is a mobile application that helps users easily locate nearby mosques and explore available Islamic services such as Quran memorization (Hifdh), youth programs, and Friday prayers.

The app combines location-based technology, a Qibla compass, and an intuitive interface to create a seamless experience for connecting with local Islamic communities.

---

## 🎯 Project Goal

To simplify access to mosques and Islamic programs by providing a modern, user-friendly digital solution that supports daily religious needs.

---

## ✨ Key Features

* 📍 Real-time mosque discovery
* 🧭 Qibla direction compass
* 📖 Quran memorization program finder
* 🔎 Smart filtering (Friday prayer, youth programs, lessons)
* 🕰️ Prayer times integration
* 🌙 Light & dark mode
* 🌍 Arabic & English support
